## Use:

- "-a, --action <type>", "choose action"
- "-i, --id <type>", "user id"
- "-n, --name <type>", "user name"
- "-e, --email <type>", "user email"
- "-p, --phone <type>", "user phone"

## Actions:

- list
- get
- add
- remove
